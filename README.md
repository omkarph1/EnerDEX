# EnerDEX 

EnerDEX is a full-stack Web3 project for peer-to-peer renewable energy trading.

The workspace has two main applications:
- `dem-project`: Solidity smart contracts with Hardhat
- `dem-frontend`: React dApp (Vite + Ethers)

## What Is Implemented

- ERC20 energy token (`ETK`) with owner minting
- Marketplace listings with token escrow
- Buy flow with exact-price ETH payment
- Platform fee collection and owner withdrawal
- Loyalty points and fee discounts (10/25/50 percent tiers)
- Frontend wallet UX, market actions, history, and income analytics

## End-to-End Local Run

1. Start blockchain node:

```bash
cd dem-project
npm install
npm run node
```

2. Deploy contracts and auto-sync frontend artifacts/env (new terminal):

```bash
cd dem-project
npm run deploy
```

3. Start frontend:

```bash
cd dem-frontend
npm install
npm run dev
```

4. Open the Vite URL, connect MetaMask to localhost, and test mint/list/buy/cancel flows.

## Integration Notes

- `dem-frontend/.env` is generated by deploy script and is currently gitignored.
- Redeploying contracts regenerates addresses and ABI copies for frontend.
- `EToken.mint` expects human token units (example: `500`), because contract scales by decimals internally.
